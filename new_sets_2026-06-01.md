## Recommended action

No ingestion is required for sets released inside the 45-day window ‚Äî
both pokedata candidates (`CRI` EN, `CBB5C` Chinese) are already in
catalog.

The five Chinese sets in the "just outside window" list above (`CSV8C`,
`30THP`, `CSVH4AC`, `CSVH4EC`, `CSVH4PC`) are the only real backlog
items. They predate this monthly probe so they wouldn't have been
caught by a previous run, but they're concrete gaps now.

If/when you decide to address the incidental gaps above, the pipeline
order is unchanged:

```
pokedata_sync.py --only <code>          # cards
pokedata_sync.py --only <code> --mirror-images
sync_pc_singles_enrich.py --tcg <name>   # PC URLs + prices
sync_sealed_products.py --tcg <name>     # sealed products
embed_catalog_rows.py --only <code>      # CLIP embeddings for scanner
```

For Magic-specific gaps (TDM), pokedata_sync.py does not handle Magic
‚Äî use whatever upstream pipeline currently feeds `mtg-*` rows.
