Standard new-set pipeline (deliberate human step — this report does not
auto-ingest):

```
pokedata_sync.py --only <code>            # cards (Pokemon-family only)
pokedata_sync.py --only <code> --mirror-images
sync_pc_singles_enrich.py --tcg <name>    # PC URLs + prices
sync_sealed_products.py --tcg <name>      # sealed products
embed_catalog_rows.py --only <code>       # CLIP embeddings for scanner
```

Magic/YGO/One Piece/Gundam singles do **not** come from `pokedata_sync.py`
— use the game's existing upstream pipeline (`enrich_from_pc_csv.py` for
YGO, the `mtg-*` feed for Magic, the OP/Gundam checklist import for those).
